JS Metodları

map - diziyi dönüştürür

find - ilk eşleşeni bulur

reduce - diziyi tek değere indirir

filter - koşula uyanları seçer

some - biri uyuyorsa true döner

every - hepsi uyuyorsa true döner

push - diziye eleman ekler

includes - eleman var mı kontrol eder

toLowerCase - harfleri küçültür

toUpperCase - harfleri büyütür

trim - baş sondaki boşlukları siler

split - yazıyı diziye böler

join - diziyi yazıya birleştirir

parseFloat - yazıyı sayıya çevirir

setInterval ve clearInterval - belirli aralıkla çalıştırır ve durdurur
