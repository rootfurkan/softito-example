# Antigravity Eğitim Materyalleri

Merhaba Öğrenciler! 🚀

Bu klasör, **Antigravity (Yapay Zeka Kodlama Asistanı)** ile modern web uygulamaları (React + Vite + Tailwind + JSON Server + Redux Toolkit) geliştirirken kullanabileceğiniz şablon ve kılavuzları içerir.

## Dosya Yapısı ve İçerik

1. **[antigravity_customization.md](file:///Users/selahaddin/Desktop/test_selo/antigravity_education/antigravity_customization.md)**:
   - Antigravity'ye özel kurallar (**Rules - `AGENTS.md`**) ve yetenekler (**Skills - `SKILL.md`**) nasıl tanımlanır, ne işe yarar detaylı açıklamaları içerir.
   - Projenizde yapay zekayı kendi geliştirme standartlarınıza nasıl uyduracağınızı gösterir.

2. **[project.md](file:///Users/selahaddin/Desktop/test_selo/antigravity_education/project.md)**:
   - Yeni bir projeye başlarken doldurmanız gereken **Proje Detay Şablonu**dur.
   - Projenizin renk paletinden, sayfa yönlendirmelerine (routing), global state mimarisinden, JSON Server veritabanı şemasına kadar tüm detayları buraya yazabilirsiniz.
   - Bu dosyayı eksiksiz doldurduğunuzda, Antigravity projeyi tamamen sizin istediğiniz detaylarda ve standartta otomatik olarak kurabilir.

## Nasıl Kullanmalısınız?

1. **Özelleştirin**: `project.md` dosyasını kopyalayın ve kendi projenizin isterlerine göre doldurun.
2. **Besleyin**: Proje dizininizde `.agents/` adında bir klasör oluşturarak `AGENTS.md` dosyasını ve `skills/` klasörünü yerleştirin.
3. **Sorun / Yönlendirin**: Antigravity'ye prompt yazarken bu dokümanları okumasını söyleyin:
   - *"Lütfen projedeki `project.md` dosyasını oku ve oradaki plana göre veritabanını ve Redux store'u kur."*
   - *"Yazacağın kodlarda `.agents/AGENTS.md` dosyasındaki kurallara sadık kal."*

Başarılar dileriz! 💻✨
